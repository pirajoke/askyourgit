# SMILE — AI Install Extension

> Adds an **"Install with AI"** button to every GitHub repo. One click to clone & set up any project with Claude Code, Cursor, Codex, or Terminal.

![Install with AI](https://img.shields.io/badge/Install_with-AI_%E2%9A%A1-blueviolet?style=for-the-badge)

---

## Install in 3 Steps

### 1. Download

[**Download ZIP**](https://github.com/pirajoke/smile/releases/latest/download/smile-extension.zip) from the latest release.

### 2. Unzip

Double-click the downloaded `smile-extension.zip` — a folder will appear.

### 3. Load into Chrome

1. Open Chrome and go to **`chrome://extensions`** (paste this into the address bar)
2. Turn on **Developer mode** (toggle in the top-right corner)
3. Click **"Load unpacked"**
4. Select the unzipped folder
5. Done! Go to any GitHub repo and you'll see the **"Install with AI"** button.

---

## How It Works

1. Go to any GitHub / GitLab / Bitbucket repo
2. Click **⚡ Install with AI** next to the Code button
3. Pick your tool: Claude Code, Cursor, Terminal, or Codex
4. Confirm and the install command runs automatically

### Features

- Smart stack detection (Node.js, Python, Rust, Go, Docker)
- Trust info (stars, license, last commit)
- NFT emoji lootbox — share repos with style via Telegram, WhatsApp, X
- Custom confirm modal before install
- Dark mode support
- Keyboard shortcut: `Cmd+Shift+I` (Mac) / `Ctrl+Shift+I` (Win)
