21st-style component system

One product, two surfaces.

The website uses a modern component layout: hero, feature cards, product mockup, download CTA, and steps. The app itself combines a menu-bar companion with an injected repo assistant inside your browser.

1

Menu-bar companion

A small macOS app stays in the top bar, installs the browser bridge, and detects current repository pages.

2

Repo page assistant

The browser extension adds Ask your GIT to GitHub, GitLab, and Bitbucket repository pages.

3

Chat before setup

Ask what the repo does, which stack it uses, and how to run it before touching your terminal.

4

Local tool handoff

Send install commands to Claude Code, Cursor, Codex, Terminal, or your own custom tools.

5

Works without a key

The prototype includes local fallback answers when hosted AI is unavailable or no API key is configured.

6

Downloadable demo

The macOS bundle, icon, native bridge, extension folder, and installer script are packaged together.

Setup

Four steps to the demo.

1

Download

Get the Ask your GIT prototype zip from this site.

2

Launch app

3

Load extension

Open chrome://extensions, enable Developer mode, and load the unzipped folder.

4

Ask a repo

Open any repo and use Ask AI to understand it before installing.
